How to Contribute

Submit a New Bill

You can help expand the Atlas by submitting new bills or legislative documents for tracking. Simply provide the URL to a legislative tracker (such as a state legislative site, city legislative site, or similar), and our system will:

  1. Check for duplicates: Verify if the bill already exists in our database
  2. Scrape and extract: Pull the bill text and key information
  3. Assess relevance: Use AI to determine if it's related to urbanist reforms
  4. Enrich with AI: Generate descriptions and categorization
  5. Add to review queue: Flag for review before publishing

Note: Processing typically takes 30-60 seconds. The system will provide real-time updates as it works through each step.

Report a New Reform or change to an existing reform

The Atlas doesn't collect data itself, but through our partner organizations. If you know of a reform that's not on the map, identify which of the partners would be good places to track this reform, then view the data in that tracker and follow that tracker's submission policy:

  1. For parking reforms: Submit to the Parking Reform Network
  2. For building code reforms: Submit to the Center for Building in North America
  3. For zoning reforms: Check if it's already tracked by Berkeley's Zoning Reform Tracker

Improve the Atlas

The code for the atlas is open source. You can contribute bug reports, bug fixes, feature requests, or new features via GitHub issues.

For more general questions reach out to the project's author at dan@austinonyourfeet.org.
